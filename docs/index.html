About this bot

CheckYerFlags is a chat bot which checks the amount of helpful flags of the users in SOBotics and posts a message to the chat room, if a user reaches or surpasses a certain amount, for example 11,111.

Due to the lack of the Stack Exchange API for getting user stats like edited posts or helpful flags, the data is read from the website. This is only done when a user actively calls the command to check the flags.

Additionally, there are also automatic checks running, but most of the bots are excluded to reduce the "scraping" to a minimum. Also, this is only done once every 30 minutes. If a user is closer than 10 flags to a new rank, this rate changes to 5 minutes (only for this user) up to the point where the user reaches the rank.

Interacting with the bot

To interact with a bot, write a message to chat with one of the alias listed below, followed by a command (example: cf alive). If the bot doesn't respond after some quickly (give at least 10 seconds) the command is most likely wrong and therefore not recognized, and if that not the case after you double-checked, it is likely dead. In that case, please ping me (@chade_) in chat and I will come and take a look when I got the time.
