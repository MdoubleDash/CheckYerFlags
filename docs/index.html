About this bot

CheckYerFlags is a chat bot which checks the amount of helpful flags of the users in SOBotics and posts a message to the chat room, if a user reaches or surpasses a certain amount, for example 11,111.

Due to the lack of the Stack Exchange API for getting user stats like edited posts or helpful flags, the data is read from the website. This is only done when a user actively calls the command to check the flags.

Additionally, there are also automatic checks running, but almost all of the bots in SOBotics are excluded to reduce the requests to a minimum. Also, this is only done once every 30 minutes, unless a user is closer than 10 flags to a new rank. If the latter is the case, this rate changes to 5 minutes (only for this user) up to the point where the user reaches the rank.

In general: If you have problems with my bot, don't hesitate to ping me. My handle in chat is @Filnor.

Interacting with the bot

To interact with a bot, write a message to chat with one of the alias listed below, followed by a command (example: cf alive). If the bot doesn't respond after some time, please ping me unless someone else already did so. I'll take a look at it as soon I got the time for it.

Running your own instance

This guide was tested on a Ubuntu 16.04 LTS server. Your experience and installation may vary on other system, especially on not Debian-based ones.

Install prerequisites

First, you need to check if your system meets all prerequisites, or if not, install them:

  • Python 3.6+
  • Git CLI
  • libxml2-dev, libxslt-dev and the python-dev packages
Recommended, but not required:
  • vim or nano
  • screen

Step 1 - Downloading the bot

After you installed the prerequisites, clone the repository:

git clone --recursive https://github.com/SOBotics/FlaggersHall.git

We use the option --recursive here, so that we get any submodules for the repository (in our case ChatExchange)

Then checkout the latest tag (check the releases page on GitHub to see what is the latest tag):

git checkout <bot version> (replace <bot version> with whatever version is the latest, e.g. v0.9.0) Now we can continue to the configuration.

Step 2 - Changing the configuration

Create a copy of the config file from the example config:

cp config.example.py config.py

Then edit the created file (you can use other editors than vi, we just use it here because it's on almost every linux system)

vi config.py

In the file you need to replace several values, but the parameters are self-explaining. Just look careful if you edit the debug or the productive config.

Step 3 - Get it up and running

If you have installed screen, you should create or attach to one now.

Run the bot with the following command:

python3.6 checkyerflags.py Depending or your system, you may need to change python3.6 to just python.

If you want to load the debug config, use this command instead:

python3.6 checkyerflags.py --debug

If everything worked out, the console output you have should look something like that:

Loading debug config...
['CheckYerFlags', 'Filnor']
Joined room 'CheckYerFlags HQ' on stackoverflow.com

And in the set chatroom a message should have been posted:

[ CheckYerFlags ] started.

The message differs if you have started in debug mode:

[ CheckYerFlags ] started in debug mode.
